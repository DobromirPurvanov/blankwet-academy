# Blankwet Academy

Видео академия за сексуално образование, интимен комфорт и модерен женски лайфстайл.

## 🌐 Live Demo

[https://academy.blankwet.eu](https://academy.blankwet.eu)

## 🎨 Brand Identity

- **Primary Color**: Purple `#6B5B95`
- **Accent Color**: Gold `#F0C675`
- **Typography**: Playfair Display (headings) + Inter (body)
- **Tone**: Empowering, educational, playful, non-judgmental

## 🏗️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React 18 + TypeScript + Vite |
| Styling | Tailwind CSS 3.4 + shadcn/ui |
| Backend | Next.js 14 (App Router) |
| Database | PostgreSQL + Prisma ORM |
| Video | Bunny.net Stream |
| Payments | Stripe |
| Email | Resend |
| Hosting | Vercel |

## 🚀 Development

```bash
npm install
npm run dev
```
